# Track and Field Olympics Visualization  

Built with SpringBoot, Java, React, TypeScript

To run:  
(SpringBoot)
cd backend  
brew install maven  
mvn spring-boot:run  

(React frontend)  
cd frontend  
npm install  
npm run dev
